### ⚠️ 스터디 규칙
* **벌금**
   * **불참** : `벌금 500원 / 회당` (최대 4회까지는 불참 혹은 과제 미제출 허용)
   * **과제 미제출** : `벌금 100원 / 문제` 
   * **지각** : `벌금 200원 / 회당` (최대 2회까지는 지각 허용)
      * 예시 : 3/2 불참 + 과제 5개 미제출 시, 벌금 500원 + 100원 * 5문제 = 1,000원

   * 스터디 종료 이후, 벌금은 스터디 참가인원 수대로 나누어 가집니다.
